海明码

By Kyaus at: 2020-11-16

海明码设计思路：

基于偶校验（“1”的个数为偶数）–> 分成多个组 –> 每组一个校验位，有多个校验位 –> 标注出错误位置

需要多少个校验位？

信息位：n个
校验位：k个~~2^k种状态
信息位+校验位=n+k –> 所以2^k种状态应包含n+k种错误情况外加一种正确情况 –> 2^k≥n+k+1
例如：n=4,则k=3，即需要添加3个校验位才能满足条件

海明码的解题步骤（例如信息位：1010）

1、确定海明码的位数

2^k≥n+k+1
n=4 –> k=3
设信息位D4D3D2D1(1010),校验位P3P2P1,对应的海明码为：H7H6H5H4H3H2H1,共七位

2、确定校验位的分布

检验位Pi放在海明码号为2^(i-1)的位置上 例如：P1P2P3 –> 1,2,4位置
信息位按顺序放到其余位置，如图：
H7 H6 H5 H4 H3 H2 H1
D4 D3 D2 P3 D1 P2 P1
1 0 1 0

3、求校验位的值

将每个信息位所处的位置用二进制表示出来
P3 P2 P1 (相当于权重)
H3: 3 –> 0 1 1
H5: 5 –> 1 0 1
H6: 6 –> 1 1 0
H7: 7 –> 1 1 1
有1的就说明此组中有这个数
P1: H3,H5,H7
P2: H3,H6,H7
P3: H5,H6,H7
异或运算来求校验位
P1 = H3⊕H5⊕H7 = D1⊕D2⊕D4 = 0
P2 = H3⊕H6⊕H7 = D1⊕D3⊕D4 = 1
P3 = H5⊕H6⊕H7 = D2⊕D3⊕D4 = 0

H7 H6 H5 H4 H3 H2 H1
D4 D3 D2 P3 D1 P2 P1
1 0 1 0 0 1 0

4、纠错

检验方程：
S1 = P1⊕D1⊕D2⊕D4
S2 = P2⊕D1⊕D3⊕D4
S3 = P3⊕D2⊕D3⊕D4
①假设接收到的是1010010(正确)
S1 = 0
S2 = 0
S3 = 0
②假设接收到的是1010000(错误)
S1 = 0
S2 = 1
S3 = 0

S3S2S1=010 –> 指明出错的位置在第二位

拓展

海明码具有1个比特位的纠错能力和2个比特位的检错能力
假如P1,P2同时出现错误，则S1=1,S2=1,S3=0,S3S2S1=011,这并不正确，不能说明是第三位错了，所以通常还需要加一个全校验位
H8 H7 H6 H5 H4 H3 H2 H1
P全 D4 D3 D2 P3 D1 P2 P1
1 1 0 1 0 0 1 0
S3S2S1 = 000且全体偶校验成功 –> 无错误
S3S2S1 ≠ 000且全体偶校验失败 –> 有一位错，纠正即可
S3S2S1 ≠ 000且全体偶校验成功 –> 有两位错，重传
